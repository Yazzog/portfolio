Mon projet professionnel

Je suis passionné d'informatique depuis mon enfance, j’aime m’informer sur les nouvelles technologies dans ce domaine et leurs actualités. J’ai par exemple choisi moi-même les composants de mon ordinateur, ce qui implique de s'informer sur les différents modèles de composants, les marques, leurs rapport performances/prix, réster à jour sur les nouveaux modèles sortis. J’aime diagnostiquer et chercher les solutions aux problèmes que j’ai pu rencontrer en informatique.

C'est donc tout naturellement que je me suis orienté vers des études d'informatique. J’avais pu découvrir la programmation durant mon année de 1ère, où j’ai pu me familiariser avec le langage Python et ainsi acquérir de bonnes bases en algorithmique. Dans le cadre de mon apprentissage j’ai pu également découvrire le développement web, avec les langages HTML et CSS, ainsi qu'acquérire des compétences en base de données avec le langage SQL. Cette année, j'ai commencé à apprendre un nouveau langage, le GO, que j'ai dévouvert dans le cadre de ma formation à l'IUT.

Mon but, après cette formation est de continuer les études dans une école d'ingénieur informatique, afin de compléter mes compétences en programmation, mais également de me spécialiser dans différents domaines.

Je vise notamment la cybersécurité, car c'est un domaine, qui me passionne, je m'informe régulièrement sur les actualités. C'est un secteur en pleine expansion, qui a donc un grand avenir devant lui.

L'intelligence artificielle est également un domaine qui m'intéresse énormément. C'est un milieu passionnant, qui possède un nombre infini d'applications possibles. L'IA est déjà présente partout dans notre vie quotidienne et le sera encore plus dans l'avenir.

Mon projet est de me spécialiser dans ces deux domaines, afin de travailler dans un des deux, et dans l'idéal, trouver un métier qui allierait les deux.
