Mon projet professionnel

Je suis passionné d'informatique depuis mon enfance, j’aime m’informer sur les nouvelles technologies en informatique et leurs actualités. J’ai notamment choisi moi-même les composants de mon ordinateur, ce qui implique de s'informer sur les différents modèles composants, les différentes marques, leurs rapport performances/prix, résté à jour sur les nouveaux modèles sortis. J’aime diagnostiquer et chercher les solutions aux problèmes que j’ai pu rencontrer en informatique.

C'est donc tout naturellement que je me suis orienté vers des études d'informatique. J’avais pu découvrir la programmation durant mon année de 1ère, où j’ai pu me familiariser avec le langage Python et ainsi acquérir de bonnes bases en algorithmique. Dans le cadre de mon apprentissage j’ai également pu découvrire le développement web, avec les langages HTML et CSS, Ainsi qu'acquérire des compétences en base de données avec le langage SQL. Cette année, j'ai commencé à apprendre un nouveau langage, le GO, que j'ai dévouvert dans le cadre de ma formation à l'IUT.
